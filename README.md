# graph of graph
A Python library of graph of graph (a decision graph for graphs)

# Quick start
```
$ git clone https://github.com/Himmelstein/graph2.git
$ cd graph2
graph2 $ make init
```

# Run test
```
graph2 $ make test
```

# Generate documentation
```
graph2 $ ./make_docu
```
Documentation is located at docs/html/
